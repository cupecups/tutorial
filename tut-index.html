1. cara update manual fw rendah ke fw 5.05:



○ download retail firmware 5.0x (bukan recovery)

○ siapkan flashdisk kosong dan format dengan format eXFAT

○ setelah file firmware sudah terdownload pastikan dengan nama "PS4UPDATE.PUP" without quotes

○ setelah itu buat folder pada flasdisk dengan urutan seperti ini ./PS4/UPDATE dan copy fw kesini

○ sudah ready, colok ke console dan matikan network connection (google it!!)

○ masuk setting - system software update

○ if following update 5.05 lanjutkan update

○ tunggu sampai update selesai lalu restart dan proses update selesai

○ untuk mengaktifkan HEN bisa akses melalui web console ke SINI atau
download esp bin firmware DISINI (tutorial upgrade esp dongle dibawah)

○ ENJOY


2. ps4 update error fix (SU-30634-6):



○ nyalakan console kemudian konek internet / pakai DONGLE

○ akses payload PS4XPLOIT buat 4.05/4.55 bisa ke DISINI via WEB BROWSER console

○ pilih payload FTP

○ saat payload FTP success akan muncul splash banner dikiri atas berisi , IP consolenya dan PORT nya

○ nyalakan PC/android/ios

○ download aplikasi FTP (saya kasih tau pakai windows download filezilla, gatau filezilla ? GOOGLE!!!!)

○ instal filezilla - new connection diatas column - isi IP console - pass dan username kosongkan - masukan PORT console

○ MISAL IP : 192.168.21.21 PORT 1337

○ PILIH QUICK CONNECT

○ buka root folder console (table kanan) buka folder "UPDATE"

○ HAPUS 2 FOLDER DENGAN NAMA "PS4UPDATE.PUP" dan "PS4UPDATE.PUP.temp.net"

○ RESTART console kemudian konek internet / pakai DONGLE

○ pilih payload "ENABLE UPDATE" tunggu samapai selesai "you are set"

○ baru lakukan update manual dari step awal pada tutorial 1


3. cara update ESP DONGLE



○ download 5.0x atau download 4.55 untuk diflash kedongle ESP

○ download 32bit atau download 64bit sesuai windows ELUH!

○ colok dongle ESP ke PC dengan kabel usb yang berfungsi

○ kalau sudah connect, buka device manager - other device - klik kanan pada usb 2.0 yang ada tanda seru

○ klik kanan pada usb 2.0 - pilih update driver - select automatically update

○ kalau pilih automatic tetep gabisa download driver bisa instal manual donwload disni pilih sesuai OS

○ RESTART PC

○ konek ulang esp ke pc

○ buka NODEMCU - pyflasher yang didownload tadi

○ pilih COM USB paling atas (com usb bisa dilihat didevice manager - other device - espxxx (COM3 atau COM4 sesuai port usb)

○ pilih ESP firmware download diatas buat 4.55 atau 5.0x

○ yang lain default aja kemudian centang "yes wipe all user data"

○ FLASH lalu coba

○ ENJOY
